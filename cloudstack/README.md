# Apache CloudStack

## Overview
Apache CloudStack is an open-source cloud management platform designed for building and managing more advanced virtualized environments.

In this project, Apache CloudStack was evaluated as a platform for an ICT learning environment running on Ubuntu Server.

## Purpose in This Project
The goal was to examine how Apache CloudStack performs in an educational setting where more advanced infrastructure management, broader administration capabilities, and stronger environment control are needed.

## Environment
The platform was deployed in a real Ubuntu Server environment and configured as part of the overall thesis project.

The evaluation focused on:
- installation and setup complexity
- infrastructure configuration
- user management
- virtual machine provisioning
- network configuration
- suitability for a shared learning environment

## Installation Perspective
Compared to MicroStack, Apache CloudStack required a more complex installation process and more preparation.

The deployment involved multiple components such as:
- Java
- MariaDB
- NFS
- KVM-related packages
- CloudStack management services

This makes the platform more demanding, but also more capable.

## Configuration Perspective
The setup process included:
- preparing the server environment
- configuring the database
- configuring storage
- setting up networking
- adding hosts and zones
- enabling virtual machine deployment

This provided a much deeper understanding of cloud platform administration.

## Strengths
- Broad management capabilities
- Better fit for larger and more advanced environments
- Stronger infrastructure control
- Suitable for more complex virtualization scenarios
- Valuable for advanced learning and administration practice

## Limitations
- More complex to install and configure
- Requires stronger technical knowledge
- Less beginner-friendly
- Higher setup and maintenance effort

## Practical Outcome
Apache CloudStack proved to be a powerful platform for more advanced learning environments.

It is especially useful when the goal is to understand deeper infrastructure management, broader virtualization control, and more complex cloud architecture concepts.
