# MicroStack

## Overview
MicroStack is a lightweight OpenStack-based platform designed for fast deployment and simple cloud environment setup.

In this project, MicroStack was used as one of the evaluated platforms for building an ICT learning environment on Ubuntu Server.

## Purpose in This Project
The goal of using MicroStack was to evaluate how well it fits an educational environment where students or multiple users need access to virtualized infrastructure in a controlled and isolated way.

## Environment
The platform was deployed on Ubuntu Server 22.04 LTS in a real server environment.

The evaluation focused on:
- installation and initial deployment
- configuration process
- user isolation
- usability
- virtual machine management
- network setup
- suitability for educational use

## Installation Perspective
One of the main strengths of MicroStack is its relatively simple deployment model compared to heavier cloud platforms.

Key observations:
- installation is straightforward
- setup is faster than more complex alternatives
- suitable for labs, demos, and learning environments
- lower entry barrier for OpenStack-based learning

## Configuration Perspective
The configuration process included preparing the node, bootstrapping the cluster, configuring networking, and validating that the environment could launch virtual machines successfully.

The work also included:
- defining network settings
- preparing access for management
- verifying services
- testing practical usability

## Strengths
- Fast deployment
- Lightweight architecture
- Good usability for introductory learning
- Practical way to study OpenStack basics
- Suitable for smaller or simpler environments

## Limitations
- Less suitable for larger production-like environments
- More limited management depth compared to Apache CloudStack
- Better for learning fundamentals than for advanced service management scenarios

## Practical Outcome
MicroStack proved to be a strong option for teaching and learning cloud fundamentals, especially where fast setup and ease of use are important.

It is particularly suitable for introductory environments where the goal is to understand virtualization, cloud services, and shared infrastructure basics.
