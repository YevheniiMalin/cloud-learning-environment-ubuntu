# Pilvipohjainen oppimisympäristö Ubuntu-palvelimella

## Yleiskuvaus
Tämä projekti esittelee Ubuntu Server -ympäristöön rakennetun pilvipohjaisen ICT-oppimisympäristön.

Projektissa tarkastellaan kahta virtualisointi- ja pilvihallinta-alustaa:
- MicroStack
- Apache CloudStack

Repositorio perustuu opinnäytetyöhöni ja muuttaa akateemisen työn tekniseksi portfoliohankkeeksi.

## Tavoitteet
- ottaa käyttöön virtualisointiympäristö Ubuntu-palvelimella
- vertailla MicroStackia ja Apache CloudStackia
- arvioida alustojen soveltuvuutta oppimisympäristöön
- tarkastella käyttäjäeristystä ja hallintaa
- dokumentoida asennus-, konfigurointi- ja testausprosessia

## Käytetyt teknologiat
- Ubuntu Server 22.04 LTS
- MicroStack
- Apache CloudStack
- KVM
- MariaDB
- NFS
- Linux-järjestelmähallinta
- virtualisointi ja verkot

## Keskeiset teemat
- virtualisointi oppimisympäristönä
- käyttäjäeristys ja monikäyttäjäympäristö
- alustojen hallinta ja käytettävyys
- verkkojen suunnittelu ja hallinta
- dokumentointi ja toistettavuus

## Tulokset
Projektin perusteella molemmat alustat soveltuvat ICT-oppimisympäristöön, mutta niiden vahvuudet ovat erilaiset.

- MicroStack sopii hyvin kevyempään ja helposti käyttöönotettavaan oppimisympäristöön
- Apache CloudStack tarjoaa laajemmat hallintaominaisuudet
- Molemmat tukevat virtuaalikoneiden hallintaa ja käyttäjäeristystä
- CloudStack soveltuu paremmin vaativampiin ympäristöihin
- MicroStack on hyvä ratkaisu perustason oppimiseen
