# Thesis Background

## Overview
This repository is based on my bachelor’s thesis project, which focused on the use of virtualization technologies as an ICT learning environment.

The original thesis explored how Ubuntu-based virtualization and cloud management platforms can support practical learning, shared infrastructure use, and platform administration in an educational context.

## Thesis Topic
**Virtualization Technology as an ICT Learning Environment**

The work focused especially on:
- MicroStack
- Apache CloudStack
- Ubuntu Server
- virtualization in education
- user isolation
- platform comparison

## Why This Repository Exists
The purpose of this repository is to transform the thesis into a more technical and portfolio-oriented project.

Instead of presenting the work only as an academic document, this repository highlights:
- technical structure
- platform comparison
- infrastructure thinking
- security and isolation considerations
- practical documentation

## Key Value of the Project
This project demonstrates:
- hands-on work with virtualized infrastructure
- ability to document technical environments
- understanding of platform differences
- practical thinking around multi-user environments
- learning-oriented infrastructure design

## Portfolio Perspective
This repository is intended to show how academic work can be turned into a practical technical portfolio project relevant to real IT infrastructure and service environments.
