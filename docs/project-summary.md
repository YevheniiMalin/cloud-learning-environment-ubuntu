# Project Summary

## Type
Infrastructure / Virtualization / Learning Environment Portfolio Project

This project is based on my bachelor’s thesis and focuses on building and evaluating an Ubuntu-based cloud learning environment using MicroStack and Apache CloudStack.

- documentation
- platform comparison

This project demonstrates:

- ability to compare technical solutions
- awareness of multi-user infrastructure requirements
- structured technical documentation skills

The project is relevant for infrastructure-focused and service-oriented IT roles because it combines technical implementation, evaluation, and documentation in one portfolio project.
